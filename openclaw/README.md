# OpenClaw Setup

## Arquivo de referencia
Use `openclaw/openclaw.json` como template do projeto.

## Instalacao no usuario (Windows)
Executar:
`powershell -ExecutionPolicy Bypass -File openclaw/install_user_config.ps1`

O script cria backup automatico se `openclaw.json` ja existir.

## Observacao
Primeiro valide o ciclo manual.
Depois ative execucao recorrente (cron/scheduler).
