# Guia de validacao de campanha

Use este checklist para classificar uma campanha:

1. Fonte oficial ou social oficial?
2. Existe screenshot util da oferta?
3. Beneficio esta claro e objetivo?
4. Existe data aparente de vigencia?
5. URL e evidencias sao rastreaveis?

Status final:
- validated
- validated_with_reservations
- review
- discarded
