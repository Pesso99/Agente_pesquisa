# Capture

## Funcao
Coletar evidencias visuais e textuais.

## Responsabilidades
- Abrir a URL
- Tirar screenshot full page
- Tirar screenshot focal se houver banner principal
- Salvar raw html quando possivel
- Salvar raw text quando util
- Registrar claims visiveis

## Regras
- Sempre salvar timestamp
- Sempre salvar URL
- Nao classificar status final
- Nao inventar informacoes nao visiveis
