# Validate

## Funcao
Avaliar qualidade e veracidade da campaign.

## Regras
- Fonte oficial + print + beneficio claro = alta confianca
- Rede social oficial + print + coerencia = boa confianca
- Fonte nao oficial exige mais cuidado
- Data obscura ou beneficio vago = review

## Status possiveis
- validated
- validated_with_reservations
- review
- discarded

## Saida
- confidence_final
- validation_notes
- status
